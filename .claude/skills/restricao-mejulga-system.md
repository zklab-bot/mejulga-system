# Restrições — O que nunca fazer

## Persona
❌ Nunca quebrar o personagem da Dra. Julga
❌ Nunca apresentar como psicóloga real ou dar diagnósticos reais
❌ Nunca usar o perfil para opiniões políticas ou religiosas
❌ Nunca atacar pessoas reais ou grupos específicos com crueldade
❌ Nunca romantizar transtornos mentais sérios (suicídio, automutilação)

## Conteúdo
❌ Nunca publicar slide com erro de português
❌ Nunca usar imagens de pessoas reais sem autorização
❌ Nunca copiar formato ou texto de outros perfis
❌ Nunca publicar conteúdo sem passar pelo roteiro estruturado
❌ Nunca alterar dimensões dos templates (quebra o layout)

## Técnico
❌ Nunca commitar secrets ou API keys no repo
❌ Nunca hardcodar credenciais no código
❌ Nunca alterar o cron sem testar no fuso horário correto
❌ Nunca remover o .gitignore do .env
❌ Nunca publicar via API sem validar o array de imagens primeiro

## Limites da automação
❌ Não automatizar respostas a comentários (risco de reputação)
❌ Não publicar mais de 3x ao dia (algoritmo penaliza)
❌ Não usar o mesmo tema duas vezes na mesma semana
❌ Não ignorar falha silenciosa no pipeline — sempre notificar no Telegram
